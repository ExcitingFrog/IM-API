Manager

API FUNCTION
Sign Up 注册管理员账号
Sign In 管理员登录
Create Department 管理员创建客服部门
Delete Department 管理员删除客服部门
Update Department 管理员更新部门信息
List Department 管理员查看所有部门信息
Create Agent 管理员创建客服
Delete Agent 管理员删除客服
Create Template 管理员创建模版信息
Delete Template 管理员删除模版信息
Update Template 管理员更新模版信息
Query Template 管理员查询模版信息
Keys 管理员获取apikey和secretkey
Access Token 获取accesstoken用于customer接口
Change Password 修改管理员密码

API FUNCTION

History Message 客服查看历史消息
Query Template 客服查询模版
Create Template 客服创建模版
Post Message 客服发送消息
Send Template 客服返送模版信息
